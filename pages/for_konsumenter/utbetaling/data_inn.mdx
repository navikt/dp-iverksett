# Data inn

## Tilleggsstønader

### Eksempel på iverksetting av ny sak
Gitt en sak med id `S-1` og en førstegangsbehandling med id `B-1` med engangsutbetaling for mai og juni. Her er hvordan en iverksetting vil se ut:

```json
{
    "sakId": "S-1",
    "behandlingId": "B-1",
    "iverksettingId": "485c17f5-2e11-49d2-b370-51120fec58d1",
    "personident": "16418245135",
    "vedtak": {
        "vedtakstidspunkt": "2024-06-03T15:24:14",
        "saksbehandlerId": "S123456",
        "beslutterId": "B123456",
        "utbetalinger": [
            {
                "beløp": "1500",
                "satstype": "ENGANGS",
                "fraOgMedDato": "2024-05-01",
                "tilOgMedDato": "2024-05-31",
                "stønadstype": "TILSYN_BARN_AAP"
            },
            {
                "beløp": "1200",
                "satstype": "ENGANGS",
                "fraOgMedDato": "2024-06-01",
                "tilOgMedDato": "2024-06-30",
                "stønadstype": "TILSYN_BARN_AAP"
            }
        ]
    }
}
```

Her er det verdt å merke seg at `iverksettingId` ikke trenger å være en UUID. Det kan være en hvilken som helst `String` med vilkårlig lengde.

### Eksempel på iverksetting av ny månedlig utbetaling

                "fraOgMedDato": "2024-05-01",
                "tilOgMedDato": "2024-05-31",
                "stønadstype": "TILSYN_BARN_AAP"
            },
            {
                "beløp": "1200",
                "satstype": "ENGANGS",
                "fraOgMedDato": "2024-06-01",
                "tilOgMedDato": "2024-06-30",
                "stønadstype": "TILSYN_BARN_AAP"

            {
