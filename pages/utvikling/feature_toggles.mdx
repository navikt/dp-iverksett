# Feature toggles

Vi bruker Unleash for å støtte feature toggles i dp-iverksett. Kontrollpanel kan nås via naisdevice og finnes [her](https://dagpenger-unleash-web.nav.cloud.nais.io/)

## Stoppe iverksetting

Vi har opprettet [kill switch](https://www.getunleash.io/blog/kill-switches-best-practice)-toggles per fagsystem for å
enkelt kunne stanse iverksetting av nye utbetalinger om det skulle bli behov for det. Når togglen er skrudd på er
iverksetting av utbetalinger for gitt fagsystem stanset i gitt miljø (dev/prod).

- [dp-iverksett.stopp-iverksetting-dagpenger](https://dagpenger-unleash-web.nav.cloud.nais.io/projects/default/features/dp-iverksett.stopp-iverksetting-dagpenger)
- [dp-iverksett.stopp-iverksetting-tilleggsstonader](https://dagpenger-unleash-web.nav.cloud.nais.io/projects/default/features/dp-iverksett.stopp-iverksetting-tilleggsstonader)
- [dp-iverksett.stopp-iverksetting-tiltakspenger](https://dagpenger-unleash-web.nav.cloud.nais.io/projects/default/features/dp-iverksett.stopp-iverksetting-tiltakspenger)
